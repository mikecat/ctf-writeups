Host

ファイル host.pcap が与えられ、通信先のWebサーバのホスト名を要求された。

host.pcap に対してstringsコマンドを実行したところ、

A file host.pcap was given and I was asked to determine the host name of the destination web server.

I applied the strings command to host.pcap, finding this line:

Host: ctf.setodanote.net

という行があり、ctf.setodanote.net がホスト名であると考えられた。

From this line, the host name should be ctf.setodanote.net.

flag{ctf.setodanote.net}
