Stegano

画像ファイルstegano.pngが与えられた。

画像をCyberChefの View Bit Plane で加工し、Render Image で表示させて調査を行った。
その結果、Colour を Red に、Bit を 4 に設定すると、左上にflag{Re4l17y_という文字列が現れた。
また、Colour を Blue に、Bit を 4 に設定すると、右下に_7h3_m1nd_rA9}という文字列が現れた。

さらに、画像の中央部にも何か書かれているようだったので、GIMPでトーンカーブをいじりながら読み取った結果、 1s_cReA73d_byと書かれているのが読みとれた。

これらの文字列を組み合わせることで、flagが得られた。

An image file stegano.png was given.

I investigated the image by processing it via "View Bit Plane" and seeing via "Render Image" on CyberChef.
As a result, I found a string flag{Re4l17y_ in the upper left when I set "Colour" to "Red" and "Bit" to "4".
Also, I found a string _7h3_m1nd_rA9} in the lower right when I set "Colour" to "Blue" and "Bit" to "4".

Moreover, I saw something seems written in the center of the image. I edited the tone curve on GIMP to read the string. As a result, I found that the string is 1s_cReA73d_by.

I obtained the flag by concatenating the strings.

flag{Re4l17y_1s_cReA73d_by_7h3_m1nd_rA9}
