Webページへのリンクが与えられた。

リンク先を開くと、

A link to a web page was given.

Opening the page, I found this text:

フラグはstardustChromeという特殊なブラウザでしか閲覧できません。

と表示された。

以下の操作によりflagが得られた。

  1. Firefoxの開発者ツールを開き、ページを再読み込みする。
  2. ファイル / のリクエストを選択し、「再送信」を押す。
  3. 「新しい要求」でUser-Agentの値を stardustChrome にし、「送信」を押す。
  4. 送信したリクエストの「応答」を見る。

I obtained the flag in this way:

  1. Open the Developer Tools on Firefox and reload the page.
  2. Select a request for the file / and click "Resend".
  3. In the "New Request" area, set the value of User-Agent to stardustChrome and click "Send".
  4. See "Response" of the request sent.
