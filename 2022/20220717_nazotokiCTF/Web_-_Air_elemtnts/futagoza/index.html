Webページへのリンクが与えられた。

リンク先を開くと、以下のものがあった。

  • パスワードが dioskouroi だという情報
  • 「Dummy Password」と書かれた入力欄
  • 「SUBMIT」と書かれたボタン

ページのソースを表示すると、以下の要素があった。

A link to a web page was given.

What is linked had these elements:

  • Information that the password is dioskouroi
  • An input field with a text "Dummy Password"
  • A button with a text "SUBMIT"

Viewing the source of the page, I found this element:

<input type="hidden" name="realPassword" value="dummyPassword">

開発者ツールでこの要素の属性 value の値を dioskouroi に書き換えてからSUBMITボタンを押すと、flagが表示された。

I changed the value of the attribute value of this element to dioskouroi using the Developer Tools and hit the SUBMIT button. As a result, the flag appeared.
