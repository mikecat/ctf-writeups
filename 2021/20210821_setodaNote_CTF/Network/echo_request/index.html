A file echo_request.pcap was given.

Opening echo_request.pcap in Wireshark, there were many "Echo (ping) request" lines.
Seeing such lines with the Length 60, I found a part in which their Data became like f, l, a, g.
I obtained the flag by reading the Data one-by-one and putting them together.
