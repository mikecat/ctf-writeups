Underscore in Corrupted

ファイル ohno.png が与えられたが、中身はPNGファイルではなさそうだった。

CyberChefを用い、ohno.png のデータと通常のPNGファイルの最初の部分 89 50 4E 47 0D 0A 1A 0A をXORしてみると、 flagの最初の部分と思われる kqctf{ が出てきた。

同じく問題名に Underscore が入っている Underscore in C (rev) で与えられたファイル Underscore_in_C.png のデータを ohno.png とXORすると、flagが得られた。

A file ohno.png was given, but its contents didn't look like a PNG file.

Using CyberChef, I applied "XOR" to the data in ohno.png with the first part of normal PNG files 89 50 4E 47 0D 0A 1A 0A.
As a result, kqctf{, which looks like the first part of the flag, appeared.

I tried applying "XOR" to the data in ohno.png with the data in Underscore_in_C.png given in the challenge "Underscore in C" (rev), which also has "Underscore" in the name.
I obtained the flag from the result.
