tippy tappies (but mobile)

音声ファイル tippytappiesbutmobile.wav が与えられた。

聴いてみると、電話のボタンを押した時のような音が入っていた。

そこで、「dtmf decoder」でググり、DTMF Decoderを見つけた。
手元の Windows 10 環境に普通にインストールしようとすると、以下のダイアログが出て拒否されてしまった。

An audio file tippytappiesbutmobile.wav was given.

Listening to the file, I found it including sounds that should appear on pressing phone buttons.

I googled "dtmf decoder" and found DTMF Decoder.
When I tried to install this tool in my Windows 10 environment as usual, it refused to install with this dialog:

This program must be run on Windows 95/98/ME/NT/2000/XP/2003 Server.

互換モードを「Windows XP (Service Pack 3)」に設定することで、インストールができた。
インストールしたツールの起動は、互換モードを設定せずにできた。

このツールで、音声ファイル中のクレジットカード番号として入力している部分をデコードし、kqctf{} で囲むことで、flagが得られた。

I succeeded to install the tool by setting the compatibility mode to "Windows XP (Service Pack 3)".
Using the tool after installation didn't require setting the compatibility mode.

I obtained the flag by decoding the part that is entered as a credit card number using this tool, and putting that into kqctf{}.

kqctf{4716097646384761}
