Just Not My Type

WebページのURLと、ファイル type.php が与えられた。

Webページにアクセスすると、Passwordの入力欄があった。
type.php を見ると、$_POST["password"] と $FLAG を strcasecmp で比較し、返り値が 0 であれば $FLAG の内容を出力する、という処理があった。

まず、Firefoxの開発者ツールを開いておき、Passwordとして a を入力して送信した。
そして、送られたリクエストの要求ボディにある password を password[] に書き換え、「編集して再送信」した。
すると、flagを含むレスポンスが得られた。

An URL of a web page and a file type.php were given.

Accessing the web page, I found a form to enter Password.
Reading type.php, I found a part that compares $_POST["password"] and $FLAG using strcasecmp, and print the value of $FLAG if the return value is 0.

Firstly, I opened the developer tools of Firefox and sent a as the Password.
Then, I changed password in the request body of the request sent to password[] and resent the request.
As a result, I got a response that contains the flag.

flag{no_way!_i_took_the_flag_out_of_the_source_before_giving_it_to_you_how_is_this_possible}
