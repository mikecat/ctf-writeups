Bad Bucket

パス部分が /index.html であるURLが与えられた。

このURLから index.html を削って / にアクセスしてみると、以下の項目を含むXMLデータが現れた。

An URL with a path /index.html was given.

I tried removing index.html from this URL and accessing to /, finding an XML data that contains this element:

<Key>buckets/.notaflag</Key>

これをもとに、同じホストのパス /buckets/.notaflag にアクセスしてみると、flagが表示された。

Seeing this, I tried accessing the path /buckets/.notaflag on the same host and the flag was displayed.
