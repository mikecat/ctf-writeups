Tera Termで接続すると、 wh47 15 y0ur n4m3 h4ck3r? と出てきたので、適当にa と入力してEnterキーを押した。(送信の改行コードはLFに設定してある)

ここでyesと入力してEnterキーを押すと、flagが出力された。

Information to connect to a TCP server was given.

Connecting using Tera Term, it printed wh47 15 y0ur n4m3 h4ck3r?.
Seeing this, I entered a as a random string and hit the Enter key. (the newline code to send is set to LF)
Then, some long sentences were printed slowly. After that, it printed 50 4r3 y0u 4c7u4lly 4 h4ck3r?.
I entered yes and hit the Enter key. Then the flag was printed.
